## 🚀 تفعيل GitHub Pages

1. **Settings** في الـrepo
2. **Pages** من القائمة الجانبية
3. **Source**: Deploy from branch
4. **Branch**: `main` / Folder: `/ (root)`
5. **Save**
6. انتظر ١-٢ دقيقة → سيظهر الموقع على `https://kaebs19.github.io/TacticaWeb/`

## 📱 الاستخدام في التطبيق
